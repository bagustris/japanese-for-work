# Easy Japanese for Work

A comprehensive collection of Japanese workplace communication lessons based on NHK WORLD-JAPAN's "Easy Japanese for Work" series.

## About This Collection

This repository contains 46 lessons covering essential Japanese phrases and communication patterns for various workplace situations. Each lesson includes:

- **Japanese phrases** with romanization and English translations
- **Dialogue examples** demonstrating real workplace scenarios
- **Grammar explanations** and usage patterns
- **Cultural notes** about Japanese workplace customs
- **Vocabulary lists** with examples
- **Keego (polite language)** patterns for professional communication

## Features of Each Lesson

Each lesson follows a comprehensive structure:

1. **Role-play Setup** - Real workplace scenario context
2. **Full Script** - Complete dialogue with Japanese, romanization, and English
3. **Key Phrase** - Main expression highlighted and explained
4. **Vocabulary** - Important words with definitions and examples
5. **Cultural Notes** - Japanese workplace customs and etiquette
6. **Grammar Points** - Detailed grammar explanations with patterns
7. **Tips** - Practical usage advice
8. **Keego (Polite Language)** - Different politeness levels
9. **Kanji** - Character readings and examples
10. **Source Reference** - Link to original NHK content

## How to Use This Resource

### For Beginners
Start with **Lessons 1-10** to learn basic workplace greetings and essential phrases.

### For Intermediate Learners
Focus on **Lessons 11-30** for professional communication and common workplace scenarios.

### For Advanced Learners
Study **Lessons 34-48** for complex situations, nuanced communication, and keego mastery.

### For Specific Situations
Use the [Full Index](index.md) to find lessons relevant to your current workplace needs.

## Study Tips

1. **Practice pronunciation** - Listen to the phrases and practice speaking them aloud
2. **Understand context** - Read the cultural notes to grasp when and how to use each phrase
3. **Master keego** - Pay attention to politeness levels appropriate for different situations
4. **Review regularly** - Revisit lessons and use the review specials (47-48) to reinforce learning
5. **Apply in practice** - Try using these phrases in real workplace situations
